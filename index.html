NFGEB is an abjad for English’s phonology I designed in 2019.

Every English consonant maps to one NFGEB character.

It also has symbols for a base-4 numbering system and a punctuation mark.

NFGEB has optional vowel diacritics, designed by Iykury. Each vowel diacritic describes a class of vowels. Vowel diacritics only appear after a consonant. Here they are applied on the consonant m.

NFGEB2025 is a variable font that implements a sans-serif typeface for NFGEB. I made it as a Unified Font Object, compiled to TTF with Google’s fontmake. It is a variable font with five axes.
